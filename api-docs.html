Lấy danh sách tất cả sản phẩm với filter, search, sort, pagination và field selection

query Parameters
page
integer >= 1
Default: 1
Example: page=1

Số trang hiện tại, mặc định là 1

limit
integer [ 1 .. 100 ]
Default: 10
Example: limit=10

Số lượng sản phẩm mỗi trang, mặc định là 10, tối đa là 100

sort
string
Example: sort=-createdAt

Sắp xếp theo field, dùng - để giảm dần, ví dụ title,-price

fields
string
Example: fields=title,descripton,price,category,subcategory,colors

Chọn các field cụ thể để trả về, cách nhau bởi dấu phẩy

keyword
string
Example: keyword=black

Tìm kiếm theo từ khóa trong title, descripton hoặc colors

price[gte]
number
Example: price[gte]=500

Lọc sản phẩm có giá từ mức này trở lên

price[lte]
number
Example: price[lte]=1000

Lọc sản phẩm có giá dưới mức này

quantity[gte]
number
Example: quantity[gte]=10

Lọc sản phẩm có số lượng từ mức này trở lên

category
string
Example: category=64df82b98c75a8c30f7a12e4

Lọc theo ID danh mục

subcategory
string
Example: subcategory=64df82b98c75a8c30f7a12e5

Lọc theo ID danh mục con

colors
string
Example: colors=Black

Lọc theo màu sắc

ratingAvg[gte]
number
Example: ratingAvg[gte]=4

Lọc sản phẩm có đánh giá trung bình từ mức này trở lên
